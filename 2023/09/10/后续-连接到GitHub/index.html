正片开始

上一次我是没想到步骤这么多,没想到要出两个,所以在前期准备中就让大家注册了GitHub账号,好了如果你和我一样把搭建博客分成了两步,那麻烦你登陆一下GitHub,因为它将是我们这一期的主角.

1. 新建一个仓库

点击New新建一个仓库

仓库名字用: 你的GitHub用户名 + github.io

比如我的就应该是

然后创建就好

2. 配置ssh key

在打开Git bush(右键展开open Git bush here)中输入

新建ssh key

按两次还是三次回车结束后会告诉你ssh key所在的地址,打开它并复制(可以打开为记事本)

打开我们GitHub

点右边你的头像,然后点击settings

点击左边的SSH and GPG keys 然后New SSH key

名字随便起 Key粘贴你刚才复制的东西 然后Add就行了
